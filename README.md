# Cognifyz Guessing Game

## Overview

The Cognifyz Guessing Game is a simple Python-based number guessing game. The program generates a random number, and the player attempts to guess it within a limited number of attempts. The game provides hints to help the player find the correct number.

## Features

* Random number generation
* User-friendly command-line interface
* Hints for higher or lower guesses
* Attempt tracking
* Input validation
* Replay option (if implemented)

## Technologies Used

* Python 3
* Random module

## How to Run

1. Make sure Python is installed on your system.
2. Open a terminal or command prompt.
3. Navigate to the project folder:

```bash
cd cognifyz_guessing_game
```

4. Run the game:

```bash
python game.py
```
